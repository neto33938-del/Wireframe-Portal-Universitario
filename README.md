Wireframe-Portal-Universitario

Bom, este é um wireframe mobile de baixa fidelidade feito no figma, onde foi criado um portal universitário de fácil acesso para o estudante visualizar sua vida acadêmica, como por exemplo Disciplinas, Notas, Atividades e Prazos. No projeto foram feitos dois caminhos para o aluno visualizar e enviar atividades de forma prática e organizada, o primeiro caminho entrando na aba de atividades e o segundo na aba de prazos.
